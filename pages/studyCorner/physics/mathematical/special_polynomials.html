Legendre Polynomial \[ {1 \over \sqrt { 1-2xt +t^2}} = \sum_{\ell=0}^\infty {P_\ell (x) t^\ell}\]
  • \[(\ell+1)P_{\ell+1} = (2\ell+1) x P_\ell -\ell P_{\ell-1}\]
  • \[ (2\ell+1) P_\ell= {d \over dx} \bigg( P_{\ell+1} - P_{\ell-1} \bigg) \]
  • \[{dP_{\ell+1} \over dx} = (\ell+1) P_\ell+x{dP_\ell \over dx} \]
  • \[ {d \over dx} \left[ (1-x^2) {dP_\ell \over dx} \right] +\ell (\ell+1) P_\ell = 0 \]
  • \[ (1-x^2) {d^2 P_\ell \over dx^2} - 2x{d P_\ell \over dx} +\ell (\ell+1) P_\ell = 0 \]
  • \[P_\ell (x)= {1 \over 2^\ell \ell !}{d^\ell \over dx^\ell} (x^2 -1)^\ell \]
  • \[P_\ell (x)= \sum_{k=0}^\infty {\ell \choose k}{\ell+k \choose k} \left(x-1 \over 2\right)^k \]
  • \[ P_0 (x) = 1\]
  • \[ P_1 (x) = x\]
  • \[ P_2 (x) = {1\over 2} (3 x^2-1)\]
  • \[ P_3 (x) = {1\over 2} (5x^3-3x)\]
  • \[ \int_{-1}^1 P_m P_n dx = {2 \over 2n +1}\delta_mn \]
  • \[ P_\ell (-x) = (-1)^\ell P_\ell(x)\]
